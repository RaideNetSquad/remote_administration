# - Система повзоляющая выполнять команды на удаленных хостах
	Состоит из трех папок
# - 1. Приложение для клиента										- Client
# - 2. Приложение для отправки JSON файла 								- JsonConf
# - 3. Приложение для сервера										- Server
# - 4. commands.json файл с командами (его надо загрузить в окно приложения JsonConf)



Структура json файла с командами
[
	{"commands":	[
				{"createfile":"путь где создавать файл"},
				{"write": {
						"where": "путь к файлу для записи туда данных",
						"text": "Ваш текст"
					}
				},
				{"copyfile": {
						"from": "путь к файлу откуда компировать",
						"to": "путь к файлу куда вставить"	
				}},
				{"delete": "путь к файлу для удаления"		
				},
				{"complete": {
					"what": "путь до исполняемой программы"
				}}
			]},
	{"sendTo":["all" или ["имя хоста1", "имя хоста2"...]]} - хосты должны быть подключенны к серверу, иначе будет ошибка!!!
]
